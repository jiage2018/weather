# 酷鸽天气---家鸽的第一个学习天气预报的程序

本程序学习了“第一行代码"里面的欧酷天气方法，并简单修改适配到最新Android Studio3.3 里面。里面使用了okhttp(3.12.0)、litepal（3.0）、gson（2.8.5)、glide（4.8.0）等开源组件，在此表示感谢！

v1.3(2019年1月17日）

1.加入天气图标等布局调整。

v1.2(2019年1月16日）

1.升级Gradle4.10.1。
2.天气预报页面添加PM2.5及AQI指数

v1.1(2019年1月14日）

1.给天气加入刷新和每隔8小时自动刷新一次天气的功能。

v1.0(2019年1月12日）

1.第一版发布。目前只能选择城市并显示未来7天天气。
